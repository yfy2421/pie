# 使用体验改进任务 🎨

## 原则

- **每一个占位符都是减分项**——搜索和 Git 必须做到"能用"级别，即使功能很简单
- **先修体验破损，再加新体验**——"搜索功能开发中" toast 比没有搜索面板更让用户失望
- **Electron 原生体验**——对话框、拖放、快捷键尽可能匹配系统行为
